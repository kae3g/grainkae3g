## 🌊 How grainflow Works

**One command. Infinite deployment.**

```bash
# Flow to all platforms simultaneously
bb flow "Your commit message"
```

This flows your work through:
1. **Build** - Incremental, fast
2. **Commit** - Append-only, grainclay
3. **GitHub** - Push and deploy Pages
4. **Codeberg** - Push and deploy Pages
5. **grainsync** - Synchronize modules
6. **grainbranch** - Update default branch + description

## 🎯 Quick Start

### **For Developers**

```bash
# Clone the grainspace
git clone https://github.com/kae3g/grainkae3g.git
cd grainkae3g

# Sync grainstore modules
bb scripts/grainstore-sync.bb sync

# Navigate modules
grain graintime    # Jump to graintime
grain-find zsh     # Find shell modules
```

### **For Contributors**

```bash
# Install grainzsh (λ prompt!)
cd grainstore/grain6pbc/teamprecision06/grainzsh
bb install
source ~/.zshrc

# Set up grainenvvars
cd ../grainenvvars
bb create-personal
bb validate
```
