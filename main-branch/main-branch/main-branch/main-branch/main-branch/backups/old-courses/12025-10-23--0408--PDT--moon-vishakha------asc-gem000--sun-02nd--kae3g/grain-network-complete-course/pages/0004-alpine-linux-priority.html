Alpine Linux embodies the Grain Network philosophy: minimal, purposeful, efficient. Like a grain of rice - small but complete, simple but powerful.

Why Alpine Linux?

1. musl libc - Clean & Secure

Unlike glibc, musl is designed for correctness and simplicity. Smaller attack surface, cleaner code, better portability. 

# musl-based static binary
ldd /bin/grain6
    statically linked

# Maximum portability achieved

2. apk - Alpine Package Keeper

Fast, efficient, no-nonsense package management. Install grain6 with a single command, minimal dependencies. 

# Install grain6 on Alpine
apk add grain6

# That's it. No apt-get update.
# No yum whatchanged. Just works.

3. Small Base Image (~5MB)

Docker containers start lean. Perfect for microservices, IoT devices, embedded systems. 88 containers fit where 1 Ubuntu would bloat. 

FROM alpine:latest
RUN apk add --no-cache grain6
CMD ["grain6", "serve"]

# Total image: ~15MB
# Ubuntu equivalent: ~200MB

4. Security-First Design

Position Independent Executables (PIE) by default. Stack-smashing protection. Regular security updates. Built for hostile environments.
