📚 Course Overview

Learn to build a complete virtualization and supervision stack from scratch:

  • 🖥️ QEMU/KVM virtualization on Ubuntu 24.04 LTS
  • 📦 NixOS 25.11 Unstable with Flakes
  • 🏠 Home Manager for declarative user environments
  • 🛡️ s6 supervision (Alpine Linux approach)
  • 🏔️ musl libc for static, secure binaries
  • 🌾 grain6 + grainwifi integration
